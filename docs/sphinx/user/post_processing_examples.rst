Post-processing Examples
========================

This section provides examples of manipulating and plotting the
post-processing data output by AMR-Wind (i.e., the files written
to the post_processing directory, not checkpoint files or plotfiles).

.. toctree::
   :glob:
   :maxdepth: 3

Simple Outputs
--------------
   
The following types of post_processing outputs produce text files
with each column labeled in a 1-line header, and due to their
simplicity, no example scripts are provided:

* FieldNorms
* Enstrophy
* KineticEnergy
* WaveEnergy
