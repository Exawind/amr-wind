Detailed Description

AMR inside an arbitrarily oriented hexahedral box

The hexahedral box is defined by three vectors xaxis, yaxis, and zaxis that specify the direction and lengths of the box with respect to origin. If the axes are aligned with Cartesian coordinates, it is better to use the CartBoxRefinement class for faster tagging.
